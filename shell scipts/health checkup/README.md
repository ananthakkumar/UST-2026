# HEALTH CHECK SCRIPT

this shell scripts provides a quick health check which includes cpu usage, memory usage, disk usage, top processes, and active network connections. 


-> reports cpu usage (user, system, idle)

-> displays memory usage in human-readable format

-> shows disk usage for all partitions

-> list top 5 processes by cpu usage.

-> displays active network connections.
